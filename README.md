# K6-perfomance-testing
Este repositório contém os scripts de teste de carga (Load Testing) desenvolvidos em k6 (JavaScript) para a pesquisa de TCC  O objetivo é medir quantitativamente o impacto da correção de vulnerabilidades de segurança (ex: IDOR, Brute Force sem Rate Limiting) na performance do backend (API REST em Go) em condições de alta concorrência.
